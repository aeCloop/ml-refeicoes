# ML Refeições

Sistema web de gerenciamento de pedidos para restaurante/delivery.

## Sobre o Projeto

O ML Refeições é uma aplicação web desenvolvida para facilitar o processo de pedidos online de refeições.

O sistema permite que clientes visualizem o cardápio, adicionem itens ao carrinho e finalizem pedidos de forma simples e rápida.

---

# Funcionalidades

✅ Listagem de refeições  
✅ Categorias de produtos  
✅ Carrinho de compras  
✅ Controle de quantidade  
✅ Cálculo automático de subtotal  
✅ Finalização de pedido  
✅ Interface responsiva  
✅ Navegação mobile

# Como Executar

Clone o repositório:

```bash

```

Entre na pasta:

```bash
cd ml-refeicoes
```

Abra o arquivo:

```txt
index.html
```

Ou utilize a extensão Live Server no VS Code.
